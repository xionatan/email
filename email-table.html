TABLE

The table module can be adapted to the desired information that's need to be displayed; More columns can be added/or removed, but for readability purposes, the maximum number of clumns should be 4 (due to the limited space available in the layout). Also, depending the on content of the columns, the width should be adjusted accordingly.
