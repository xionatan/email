FOOTER

Since almost all e-mails that Tradeshift sends out have an action button so that the user can take an action, in the footer of all e-mails we should put the link (in clear text) of that action button (due to the fact that some e-mail clients will block the action on the button - security policy - so thous the user can still pe able to perform the action by copy-pasting the link in the browser.
If the e-mail has no action button, then the footer should NOT be included in the template!
